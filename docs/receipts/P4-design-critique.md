- **(c) broadsheet hairline-rule grid** — ABSENT. The only rules are dossier-row
  separators inside one galley column; layout is a single typeset column floating over
  the field, not a multi-column ruled grid.

manifests against wide engraved eyebrows is the recognizable typographic voice.
